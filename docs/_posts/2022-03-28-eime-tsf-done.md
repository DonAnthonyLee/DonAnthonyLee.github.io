注：服务端及客户端目前均支持 x64 架构 Windows XP ~ Windows 11。

# 免费试用版输入功能

**目前免费试用版与收费版无异，且不设时限。**

商店收费版本功能以后会逐渐加入，目前设置较高收费的目的是杜绝有些人不明就里进行购买。无论如何，商店免费试用版将保持与其他平台免费版（比如 EIME-XIM 等）基本同等功能。
