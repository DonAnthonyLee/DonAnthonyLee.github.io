近两天 EIME-TSF 调试并基本完成既定目标，比原先预计晚了将近一个月。目前服务端通过商店应用程序方式进行展现；客户端因为目前 Windows 10/11 系统尚未支持，需要再行人工进行组件注册。

注：服务端及客户端目前均支持 x64 架构 Windows XP ~ Windows 11。


我已提交至 Microsoft 应用商店进行审核，不论结果如何，此项工作暂且告一段落了。
